## Overview
Empowered Safety is a web-based emergency response system designed to help women trigger SOS alerts quickly using voice commands. The system sends emergency messages with location details to registered contacts and supports real-time monitoring.

## Features
- Add multiple emergency contacts
- Voice-activated SOS trigger
- Location sharing using GPS
- Twilio SMS integration via Flask backend
- Health monitoring using PPG signal detection
- Heart rate calculation using sliding window algorithm

## Technologies Used
- HTML
- CSS
- JavaScript
- Python (Flask)
- Twilio API
- GitHub Pages
